# 🕊️ 鸽子提醒器

一个有趣的工具，用来幽默地提醒那些约好参与游戏却迟迟不露面的"鸽子"朋友们。通过光速计算将等待时间转换为天体距离，生成专属的太空主题明信片。

## ✨ 功能特色

### 🚀 核心功能
- **智能计算**：根据等待时间乘以光速，计算对应的天体距离
- **天体匹配**：从18个天体数据库中找到最接近的天体（月球到可观测宇宙边缘）
- **幽默信件**：自动生成3种不同风格的提醒信，措辞风趣幽默
- **互动明信片**：生成包含天体信息和可点击按钮的HTML明信片

### 🎨 明信片功能
- **精美设计**：太空主题背景，星空动画效果
- **分区布局**：
  - 📋 标题区域：项目名称和介绍
  - 🌟 天体展示区域：天体信息和动画效果
  - 📧 信件内容区域：完整的幽默提醒信
  - 🚀 交互按钮区域：病毒式传播按钮

### 🌐 分享传播
- **HTML页面下载**：生成完整的独立HTML文件
- **链接分享**：支持原生分享API和链接复制
- **病毒式传播**：每个明信片都包含"生成我的明信片"按钮

## 🛠️ 技术实现

### 前端技术
- **纯HTML/CSS/JavaScript**：无依赖，开箱即用
- **响应式设计**：适配各种屏幕尺寸
- **Canvas绘图**：AI风格的天体和飞船绘制
- **动画效果**：CSS动画和JavaScript交互

### 核心算法
- **光速计算**：等待时间 × 光速 = 传播距离
- **天体匹配**：最小差值算法找到最接近的天体
- **距离格式化**：自动转换为合适的单位（光秒/光分/光时/光天/光年）

## 📊 天体数据库

包含18个天体，涵盖：
- **太阳系天体**：月球、太阳、八大行星
- **恒星系统**：比邻星、半人马座α星、天狼星等
- **星系结构**：银河系中心、仙女座星系、室女座星系团
- **宇宙边界**：可观测宇宙边缘

## 🎯 使用方法

1. **输入信息**
   - 朋友的名字
   - 游戏名称
   - 等待时间（小时、分钟、秒）

2. **生成结果**
   - 点击"🚀 计算天体位置并生成提醒信"
   - 查看匹配的天体和幽默信件

3. **创建明信片**
   - 点击"🖼️ 生成明信片"
   - 获得精美的HTML明信片

4. **分享传播**
   - 下载HTML文件或分享链接
   - 朋友点击明信片按钮可生成自己的版本
